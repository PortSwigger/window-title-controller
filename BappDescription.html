This is an extension for Burp Suite that allows you to take control of your Burp Suite window title. It was built using the Montoya API and enhances your ability to customize the Burp Suite window title beyond its default display.

Features

  • Displays or hides the - licensed to ... suffix in the Burp Suite window title
  • Allows you to set a custom window title of your choice
  • Lets you reset back to the default Burp Suite window title
  • Displays the current window title in the Title menu

Usage

Upon loading the extension, a new Title menu will appear in the Burp Suite menu bar. From this menu you can:

  • Check the current window title with Current: ...
  • Toggle the visibility of the license suffix
  • Set a custom window title using a dialog prompt
  • Reset the window title back to its Burp Suite default

An acknowledgement to ronen1ns Burp License Title Cleaner, which inspired this project. This enhances the original with more features such as custom title setting and reset functionality.
